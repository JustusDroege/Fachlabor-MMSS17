\chapter{Actual designing and FEM-Simulation}
\label{sec:FEM-Simulation}
In diesem Kapitel wird zunächst ein CAD-Modell von verschiedenen Sensordesigns mit variablen Abmessungen erstellt. Dazu wird die Software SolidWorks verwendet. Im Anschluss werden verschiedene Simulationen durchgeführt. 

\section{Sensordesign (Justus)}
\label{sensordesign}
Zur Auswahl stehen zwei Designs, die beide für eine Druckmessung geeignet sind. Zum einen eine Anordnung mit Rahmen und einfacher Membran, zum anderen ein Design mit zusätzlich auf der Membran platziertem Boss. Dabei gelten die folgenden Abmessungen.
\begin{itemize}
\item Abmessung Grundfläche (Siliziumwafer) beträgt \unit[$6500*6500$]{$\mu$m}
\item Dicke des Siliziumwafers beträgt 450$\mu$m
\item Der fertigungsbedingte Winkel beträgt 54,7$^\circ$
\end{itemize}
Dazu sollen für das Design mit Boss folgende variable Abmessungen gelten(vgl. Tabelle \ref{tab:variabel}).

Um das gewünschte Design zu modellieren, wird folgendermaßen vorgegangen: Zunächst wird die Einheit in SolidWorks von Milimeter auf Mikrometer geändert. Nun kann die Grundfläche, also der Siliziumwafer, skizziert werden. Mit einer Extrusion erreicht man die gewünschte Dicke des Wafers. Anschließend wird die Membran auf dem Wafer skizziert. Dabei hilft es, ein Quadrat über den Mittelpunkt des Wafers zu ziehen. Um den Winkel von 54,7$^\circ$ einzuhalten, muss nun beim extrudieren ein Verjüngungswinkel eingestellt werden. Ist dieser Schritt durchgeführt, kann die Bossstruktur mit gleichem Vorgehen eingefügt werden. \\
Mit Hilfe der Tabellenfunktion von SolidWorks können Dimensionierungsparameter der Designs in einer Exceltabelle gespeichert und auf Wunsch angepasst werden. Diese Funktion wird dazu benutzt, um die in Tabelle \ref{tab:variabel} vorgestellten Abmessungen durch einen einfachen Befehl auszuwählen und darzustellen. \\
Für die FEM-Simulation werden die beiden grundlegenden Sensordesigns geviertelt, um durch das Ausnutzen der Symmetrie Rechenzeit zu sparen.
\section{FEM-Simulation (Justus)}
\label{sec:FEM}

\begin{itemize}

\subsection{Kritische Betrachtung (Justus)}
\label{kritische betrachtung}
Eine FEM-Simulation wird meistens unter idealen Bedingungen durchgeführt, da sich in der Realität auftretende Störeinflüsse schlecht simulieren lassen. Zunächst gibt es, gerade bei Mikrosystemen, welche im häufigsten Fall durch Ätzen hergestellt werden, stets Fertigungstoleranzen. Diese beeinflussen nicht die Simulation, aber ändern das Verhalten des realen Bauteils. Hinzu kommt, dass das Silizium, das für die Simulation verwendet wurde, als isotroper Werkstoff simuliert wird. Dies bedeutet, dass das Materialverhalten in alle Raumrichtungen gleich ist. In der Realität ist Silizium jedoch, unter anderem bedingt durch seine monokristalline Struktur, anisotrop. Anisotropie bedeutet die Richtungsabhängigkeit des Materialverhaltens, die den Rechenaufwand einer Simulation enorm vergrößern würde. Ein Beispiel für die Verwendung von Anisotropie in Bezug auf Silizium ist die Berücksichtigung der Miller'schen Indizes beim Ätzprozess. Als weiterer Grund für eine Abweichung der FEM-Simulation zur Realität ist die gewählte Elementgröße von 60 bzw. 120 $\mu$m. Es besteht zwar die Möglichkeit, die Vorgaben weiter zu verfeinern, jedoch bleiben steht Bereiche, auf denen interpoliert werden muss. Außerdem vergrößert sich bei genaueren Analysen die Rechenzeit enorm.\\
Abschließend kann behauptet werden, die durchgeführten Simulationen sind lediglich eine grobe Annäherung an das reale Systemverhalten.
